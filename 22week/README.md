# Week 22 Implementations

This workspace contains seven completed implementations for vector search, FAISS retrieval, chunk optimization, semantic retrieval, a mini RAG system, RAG performance analysis, and RAG failure analysis.

## Structure

- Implementation 1: Mini Vector DB — Top-K cosine search
- Implementation 2: FAISS Retrieval — FAISS/NumPy Top-K search
- Implementation 3: Chunk Optimizer — compare chunk sizes
- Implementation 4: Semantic Retrieval — embeddings + ranking
- Implementation 5: Mini RAG — retrieval + generation pipeline
- Implementation 6: Performance Analysis — compare params/models
- Implementation 7: Failure Analysis — retrieval/context/knowledge
